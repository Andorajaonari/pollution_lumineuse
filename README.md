# TESS-W Night Sky Brightness Pipeline -- Maido Observatory (2022-2025)

This repository contains the complete data-processing pipeline used to analyse 34 months of night sky brightness (NSB) measurements acquired by a TESS-W photometer at the Maido atmospheric observatory (La Reunion Island, 2200 m a.s.l.). The pipeline implements rigorous filtering of lunar, galactic, cloud, and lidar contamination, aggregates high-quality nights, and produces the statistical analyses and figures reported in the associated publication.

## Repository structure

- `tess_pipeline.ipynb` : main notebook that runs the full analysis (loading, filtering, aggregation, trend analysis, correlations, visualisations).
- `aggregated_nights.csv` : table of the 247 quality nights retained after aggregation.
- `figures/` : output figures (PNG format) generated by the notebook.
- `disponibilite_mensuelle.csv` : monthly data availability table.
- `climatologie_mensuelle.csv` : monthly NSB climatology.
- `indicateurs_annuels.csv` : annual NSB statistics.
- `nuits_extremes_par_mois.csv` : extreme nights (brightest/darkest) per month.

## Dependencies

The pipeline requires Python 3.8+ and the following packages :

- numpy
- pandas
- matplotlib
- seaborn
- scipy
- scikit-learn
- statsmodels
- pymannkendall
- pytz

Install them with `pip install -r requirements.txt` (a requirements file is provided).

## Usage

1. Place the raw TESS-W CSV file (`tess_data.csv`) and the F10.7 flux table (`F107_fluxtable.txt`) in the repository root.
2. Open `tess_pipeline.ipynb` in Jupyter Lab or Google Colab.
3. Adjust the parameters in the first code cell (timezone, paths, thresholds) if necessary.
4. Run all cells. The notebook will produce all figures and export the aggregated data.
